# myroutetoaiengineer

Learning Math and Python.

## Setup

This project uses a virtual environment with numpy installed.

### To run scripts:

**Option 1: Activate the virtual environment first**
```bash
source venv/bin/activate
python math.py
```

**Option 2: Use the venv's Python directly**
```bash
./venv/bin/python math.py
```

### Note:
If you get `ModuleNotFoundError: No module named 'numpy'`, make sure you're using the virtual environment's Python interpreter, not the system Python.
